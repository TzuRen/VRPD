# VRPD

**To compile:**

Make sure you have installed cmake, and run the scripts:
~~~
cmake cmakelist
make
~~~
The execuable file "vrpd" is generated in ./bin

**To use:**

The [vrpd][null-link] takes a couple of arguments. For more details, please run
~~~
./bin/vrpd -?
~~~
